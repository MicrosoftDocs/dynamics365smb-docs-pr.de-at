Das Fibu Buch.-Blatt zeigt nur eine begrenzte Anzahl Felder in der Buch.-Blattzeile standardmäßig an. Wenn Sie die anderen Felder sehen wollen, wie das Feld **Kontoart**, wählen Sie die **Zeigen Sie mehrere Spalten an** Aktion aus. Um die zusätzlichen Felder erneut auszublenden, wählen Sie **Zeigen Sie weniger Spalten an**.
